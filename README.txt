tagpoll README
==================

I wanted to do a poll where the respondent needed to pick X choices from a
field of maybe 5*X alternatives. I couldn't find any existing polling
solution. So I wrote one.

Getting Started
---------------

This project has been developed under Python 3.3. It will likely fail
under any earlier version.

- cd <directory containing this file>

- $venv/bin/pip install -e .

- or:

- $venv/bin/python setup.py install

- $venv/bin/initialize_tagpoll_db development.ini

- $venv/bin/pserve development.ini
